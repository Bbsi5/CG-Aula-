AR - Hit and Build


This example uses WebXR to add virtual objects without the use of fiducial markers.

The example is best suited for use with mobile devices. After clicking on the "Open" button below, press the "Start VR" button. This button is only available if your mobile device is ready for WebXR. Try moving your phone and focusing on flat objects around you, such as on the floor, a table, etc. When a white rectangle is visible, you can click on your phone's screen to add an object.

Note: To test this example on a computer you have to install the WebXR API Emulator extension and select the "Samsung Galaxy S8+" device.
